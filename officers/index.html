Government NGO Tracking

Search through nonprofit filings to find organizations and their principal officers. Data includes EINs, organization names, and officer details.

Below is a simple two-part keyword search over nonprofit filings.

Nonprofit search looks up by EIN or nonprofit name.
Officer search looks up by principal officer name.

Each typed word is treated as an AND condition (i.e., all must match).

This search runs in two passes per keyword:

  1. exact keyword match
  2. prefix matches (i.e., other keywords that start with that string)

For performance reasons, only the first 100 matching rows are displayed.
